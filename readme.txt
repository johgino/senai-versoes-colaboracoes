Etapas da criação do versionamento de projeto 
- criei a pasta atividade .
- usei o GIT BASH HERE com o botão direito do mouse para iniciar o consele do GIT.
- Criei o arquivo de texto "readme.txt" onde irei especificar o caminho que utilizei e os principais comandos.
- usei o git add . para adiconar o arquivo a fila.
- usei o git status para verificar se o arquivo havia sido adicionado a fila ou se havia algum erro.
- visto que nao havia nenhum erro usei o git commit -m " inicio da atividade" para criar a primeira versao do meu projeto.
